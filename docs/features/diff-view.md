# Diff View

## What it is
The main reading surface.

## What it does
- Renders the diff as hunks instead of one long patch blob.
- Keeps the current cursor position visible.
- Marks visited lines so the reviewer can see what has and has not been read.
- Surfaces AI and comment state inline with glyphs instead of burying it in a side panel.
- Shows hunk metadata like coverage, symbol definitions, references, and teammate badges in-place.
