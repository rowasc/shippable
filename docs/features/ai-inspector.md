# AI Inspector

## What it is
Granular review context.

## What it does
- Aggregates AI concerns for the current hunk in one place.
- Shows severity, summary, detail, ack state, and replies per note.
- Shows hunk-level AI summary when present.
- Shows teammate verdicts and replies in the same inspection surface.
- Keeps user comments beside the AI context instead of splitting them into a separate mode.
