# Line Comments And Replies

## What it is
Threaded discussion on top of individual lines and AI notes.

## What it does
- Lets the reviewer reply directly to an AI note.
- Lets the reviewer start their own line comment thread.
- Keeps existing replies attached to the exact line they belong to.
- Supports draft persistence so closing the composer does not lose in-progress text.
- Lets the reviewer clean up their own reply history if a thread is no longer useful.
