# Review Progress

## What it is
The app's progress model for a review session.

## What it does
- Tracks which lines the cursor has actually visited.
- Rolls that up into file-level read percentages.
- Separates passive reading from explicit file sign-off.
- Shows overall reviewed-file count in the status bar.
- Keeps progress visible in the sidebar, diff header, diff body, and footer at the same time.
