# AI Annotations

## What it is
The metadata layer that lets the diff carry machine and teammate review signals.

## What it does
- Attaches per-line AI notes with severity, summary, detail, and optional verification recipe.
- Supports hunk-level AI summaries.
- Supports teammate verdicts on hunks.
- Uses stable reply-key shapes so note replies, teammate replies, line comments, and block comments can all live in one reply store.
